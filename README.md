Assignment Summary:
In this assignment, the Pandas library was imported and the dataset was loaded from a CSV file into a DataFrame using read_csv(). The dataset was explored using head(), tail(), shape, columns, dtypes, and info() to understand its structure, dimensions, column names, and data types. Missing values were identified using isnull().sum() and handled using fillna() by replacing missing values in numerical columns with their median values. The dropna() function was also demonstrated to remove rows containing missing data. Basic data operations were performed by selecting specific columns and filtering rows based on conditions. Duplicate records were identified using duplicated().sum() and removed using drop_duplicates() to ensure data quality. A new derived column, total_amount, was created by multiplying the price and quantity columns. Finally, the cleaned dataset was saved as cleaned_sales_data.csv using to_csv() for future analysis and use.
